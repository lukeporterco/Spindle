# Spindle Docs

This folder is the project memory for Spindle.

- `architecture/` records internal architecture, pass history, arc state, capability boundaries, non-goals, and handoffs.
- `mods/` records developer-facing behavior for future mod authors and pack authors.

Architecture docs should prevent drift. A reader should be able to reconstruct what each pass added, what it did not add, what reports or APIs it affected, and what future work is allowed to build on it.
